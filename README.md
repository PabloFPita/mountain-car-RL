## Introduction

Flappy Bird RL is an implementation of the Flappy Bird game using reinforcement learning techniques. The goal is to train an AI agent to navigate through the pipes by learning from its mistakes and improving its performance over time.

## Installation

To install and run the project, follow these steps:

1. Clone the repository:

    ```shell
    git clone https://github.com/your-username/flappy-bird-rl.git
    ```

2. Install the required dependencies:

    ```shell
    pip install -r requirements.txt
    ```

## Usage

To play the game manually, run the following command:
 ```shell
    python flappy_bird.py
 ```
